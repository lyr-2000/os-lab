## 显示文字到屏幕

```asm
mov ax,0xb800;   指向屏幕文本模式的显示缓冲区
mov es,ax; es 是段寄存器

mov byte [es:0x00],'I'
mov byte [es:0x00],0x07
jmp $
times 510 - ($-$$) db 0
 ; 为什么是 510 不是512呢？ 因为最后2个字要填充魔数 ，0xaa55
DW 0XAA55    



```
